Требования к работоспособности "приложения":

1. Наличие интернета.
2. Установленный NodeJS.

Шаги для использования "приложения":

1. Открыть консоль в корневой директории "приложения".
2. Запустить сервер (написать команду в консоле: >node js/server.js).
3. Постучаться на сервер - localhost:8000 .
4. Подождать пока достанет инфу из базы данных (так как она удаленно на mongolab находится).
5. Наслаждаться.
